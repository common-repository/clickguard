== Installation ==

In order to use the WordPress plugin, you must have an active ClickGUARD account. You can register for an account [here](https://www.clickguard.com/start-free-trial/). Terms of service are available [here](https://www.clickguard.com/terms-of-service/).

= Installation steps =

1. Upload 'clickguard' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings -> ClickGUARD to open the settings
1. Link your site with ClickGUARD
1. Configure tracking and dashboard information
